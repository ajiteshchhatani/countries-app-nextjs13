## Getting Started

To run this repo locally, clone the project.

Then, install all dependecies using

```
npm install
# or
yarn
# or
pnpm install
```

 Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
